## 🛠️ Logiciels et Outils Utilisés
 
### Environnement de développement
 
| Logiciel | Version | Rôle | Téléchargement |
|---|---|---|---|
| **WAMP64** | 3.4.0 | Serveur local (Apache + MySQL + PHP) | [wampserver.com](https://www.wampserver.com) |
| **PHP** | 8.4.0 | Langage backend | [php.net/downloads](https://www.php.net/downloads.php) |
| **MySQL** | 8.4.7 | Base de données | [mysql.com/downloads](https://dev.mysql.com/downloads/mysql/) |
| **Apache** | 2.4.x | Serveur web local | [httpd.apache.org](https://httpd.apache.org/download.cgi) |
 
> 💡 **WAMP64 inclut déjà PHP, MySQL et Apache** — un seul téléchargement suffit.
 
### Éditeur et outils de code
 
| Logiciel | Rôle | Téléchargement |
|---|---|---|
| **Visual Studio Code** | Éditeur de code principal | [code.visualstudio.com](https://code.visualstudio.com) |
| **phpMyAdmin** | Interface de gestion de la base de données | [phpmyadmin.net](https://www.phpmyadmin.net/downloads/) |
| **Git** | Gestion de versions | [git-scm.com](https://git-scm.com/downloads) |
